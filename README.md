# 0xhack-integration-workshop

This is an example repository showing how to add Ramp to your JavaScript or TypeScritp based app.

Relevant links:
- [https://docs.ramp.network/](https://docs.ramp.network/)
- [Ramp Discord](https://discordapp.com/invite/qv8Zprw)

Start by running

```sh
git clone git@github.com:RampNetwork/0xhack-integration-workshop.git

yarn && yarn start

# or

npm install && npm start

```
